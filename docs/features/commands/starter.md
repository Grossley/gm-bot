# Starter

> **Matches:** `!starter`, `!starterpack`, `!starterkit`

> **Usable by:** Everyone

Sends the user a list of helpful resources.

> **Matches:** `!starter`, `!starterpack`, `!starterkit`

> **Usable by:** Staff & Ducks
 
Sends a specific user a list of helpful resources

```
!starter [@user]
```
Sends `[@user]` a list of resources

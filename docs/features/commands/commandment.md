# Commandment

> **Matches:** `!commandment`, `!rtfm`

> **Usable by:** Everyone

Posts a link to a given commandment from Pixelated Pope's [commandment video](https://www.youtube.com/watch?v=G1WxKEk6Wrw).

```
!commandment [roman numeral]
```
Posts the given commandment
```
!rtfm
```
Posts commandment I

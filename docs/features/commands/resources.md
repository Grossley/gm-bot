# Resources

> **Matches:** `!resources`

> **Usable by:** Everyone

Sends the user a list of helpful links to online GameMaker resources.

> **Matches:** `!resources`

> **Usable by:** Staff & Ducks
 
Sends a specific user a list of helpful links to online GameMaker resources

```
!resources [@user]
```
Sends `[@user]` a list of resources
